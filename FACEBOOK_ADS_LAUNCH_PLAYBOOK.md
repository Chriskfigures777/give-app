# The Exchange App — Facebook Ads Launch Playbook
### Initial Launch Ads + Retargeting Sequences + A/B Test Variants

---

> **Primary launch focus: The Website Builder**
> Per the marketing plan, the #1 ad hook is "build your church website in 10 minutes, free 14-day trial, no credit card required." This is tangible, immediate, and zero-risk for any church admin who has been putting off their website.

---

## Campaign Architecture Overview

```
COLD TRAFFIC CAMPAIGNS (Phase 1 — $1,000/mo test budget)
  ├── Campaign A: Website Builder Hook
  ├── Campaign B: Free Giving Platform Hook
  └── Campaign C: All-In-One Platform Hook (broader)

RETARGETING CAMPAIGNS (Phase 2 — add at Month 4–5)
  ├── RT-1: Visited landing page, did NOT sign up (7-day window)
  ├── RT-2: Signed up free, did NOT start trial (14-day window)
  └── RT-3: Started trial, did NOT convert to paid (7-day window)
```

## PART 3 — A/B TEST SCHEDULE

> Run ONE variable at a time. Do not test multiple things simultaneously — you won't know what caused the result.

---

### Test Phase 1 (Month 2–3): Hook Copy
Run all three at $333/mo each ($1,000 total). Winner = lowest CPL.

| Version | Headline |
|---|---|
| **A** | `Start your church website free` |
| **B** | `The all-in-one platform your church has been missing` |
| **C** | `Run your church's giving, website, and budget from one place` |

**Decision point:** After 2 weeks, pause the two lower performers. Double down on the winner.

---

### Test Phase 2 (Month 3): Creative Format
Take the winning hook. Test three creative formats.

| Version | Format |
|---|---|
| **A** | 30-second testimonial video (pilot church pastor on camera) |
| **B** | 90-second screen recording of website builder being used |
| **C** | Static image: `"380,000 US churches. Most use 5+ disconnected tools."` |

**Decision point:** After 2 weeks, pause lower performers. Run winner at full $1K/mo budget.

---

### Test Phase 3 (Month 3–4): Audience Targeting
Take winning hook + winning creative. Test three audiences.

| Version | Audience |
|---|---|
| **A** | Job titles: Pastor, Executive Pastor, Church Administrator, Ministry Director |
| **B** | Interests: Church administration, nonprofit management, Christian ministry |
| **C** | Broad: 25–55, US, no interest targeting (let Meta's algorithm find the audience) |

**Decision point:** After 3 weeks, cut the two underperformers. Scale the winner.

### Test Phase 4 (Month 4): CTA / Landing Page
Take the proven ad. Test the call to action.

| Version | CTA |
|---|---|

## PART 4 — AD CREATIVE SPECS & PRODUCTION NOTES

### Video Ad Specs (Facebook/Instagram)
- Aspect ratio: 4:5 (best for feed) or 9:16 (Reels/Stories)
- Length: 15–30 seconds for cold traffic; up to 60s for retargeting
- Captions: REQUIRED — 85% of videos watched without sound
- Hook: First 3 seconds must stop the scroll. Open with the problem, not the logo.
- Branding: Exchange logo in corner, not first frame
- CTA overlay: Show text CTA on screen in final 5 seconds

### Static Image Ad Specs
- Size: 1080x1080 (square) or 1080x1350 (portrait — 20% more screen space)
- Text rule: Keep text under 20% of image or reach will be reduced by Meta
- Font: Clean, readable. No decorative fonts on mobile.
- Background: Dark or branded — avoid white backgrounds that blend into feed

### Screen Recording / Demo Video
- Record at 2x or higher resolution (1440p if possible)
- Highlight cursor movements — use a cursor highlighter tool
- Add zoom-ins on key actions (clicking "Build Website", entering a domain, etc.)
- Voiceover: Conversational, not scripted-sounding. One clear sentence per action shown.

## PART 6 — FACEBOOK PIXEL & TRACKING SETUP

> These events must be firing BEFORE ads launch. No pixel = no optimization = wasted money.

**Required Pixel Events:**
1. `PageView` — All pages (standard — install globally)
2. `ViewContent` — Landing page / pricing page view
3. `Lead` — Free account signup completed
4. `StartTrial` — Custom event: user clicks "Start 14-day Trial"
5. `Purchase` — Custom event: paid plan activated (pass plan name + value)

**Conversion API (CAPI):** Set up server-side events via Supabase webhook → Facebook CAPI. This is critical post-iOS 14 — browser pixel alone misses 30–40% of conversions.

**UTM Parameters for all ads:**
```
?utm_source=facebook&utm_medium=paid_social&utm_campaign=[campaign_name]&utm_content=[ad_name]
```

## PART 7 — PERFORMANCE BENCHMARKS

> These are the go/no-go thresholds. If an ad set is not hitting these after 7–14 days and $100+ in spend, pause it and move budget to the winner.

| Metric | Pause Threshold | Target |
|---|---|---|
| Click-Through Rate (CTR) | Below 1.0% | 2.0%+ |
| Cost Per Click (CPC) | Above $3.00 | Under $1.50 |
| Cost Per Lead (CPL) | Above $80 | Under $40 |
| Trial Start Rate (of signups) | Below 15% | 30%+ |
| Trial-to-Paid Conversion | Below 15% | 25%+ |
| Landing Page Conversion Rate | Below 5% | 10–15% |

**Weekly review every Monday.** Check these numbers. Kill losers fast. Scale winners.
